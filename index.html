Projeto Website

Este projeto simula uma página de notícias, com um layout moderno e responsivo. A ideia é oferecer uma experiência fluida para os usuários, permitindo fácil navegação entre as matérias e as notícias.

🔹 Tecnologias utilizadas: HTML, CSS, JavaScript

✔️ Layout responsivo para diferentes dispositivos

✔️ Menu de navegação interativo

✔️ Design simples e intuitivo

🔹 Funcionalidades:

A página contém um menu fixo na parte superior da tela, com opções que exibem uma caixa com os itens quando passamos o mouse sobre elas ou as selecionamos.

A estrutura é dividida em duas colunas: uma para as matérias e outra para as notícias. A coluna de notícias possui rolagem independente, permitindo uma navegação fluida. Além disso, há pequenos elementos estilizados, como uma imagem com fundo desfocado, uma propaganda com bordas animadas e um banner fixo ao lado do conteúdo assim que a coluna de notícias chega ao fim.

O site também conta com alguns jogos interativos: um jogo para adivinhar o número escolhido pelo computador, com diferentes níveis de dificuldade, um jogo da velha e um jogo baseado no clássico Missile Command, também com seleção de dificuldade.

📌 Conclusão:

Esse projeto foi desenvolvido com o objetivo de explorar diferentes técnicas de layout, interatividade e estilos visuais, criando uma experiência dinâmica para o usuário. Além do design funcional, os jogos adicionam um toque de entretenimento, tornando a página mais envolvente. Continuarei aprimorando o projeto, testando novas ideias e aperfeiçoando suas funcionalidades.

🔗 Acesse o projeto https://williamalvesoliveira.github.io/website/

Projeto Dispensa

🔹 Python, CustomTkinter

✔️ sistema de validação de dados

✔️ Design simples e intuitivo

O projeto Gestor de Dispensa, foi desenvolvido com o objetivo de ajudar na organização e controle de itens de uma dispensa doméstica. O app permite aos usuários cadastrar produtos, definir quantidades e controlar quando é necessário reabastecer itens essenciais.

🔹 Funcionalidades:

A interface gráfica, desenvolvida com a biblioteca CustomTkinter, proporciona uma navegação simples e intuitiva, com uma visão clara de todos os itens cadastrados na dispensa.

O app permite adicionar, editar e excluir produtos facilmente. Para cada item, é possível definir a quantidade disponível, a categoria (essencial ou não) e o período de compra recomendado, com alertas automáticos para avisar quando o item precisa ser reabastecido.

A funcionalidade de "Mostrar itens" exibe os produtos disponíveis na despensa e seus respectivos estoques. As quantidades aparecem destacadas em verde quando estão dentro do nível desejado, e em vermelho quando estão abaixo, indicando que precisam ser reabastecidas.

Além disso, a aplicação oferece a possibilidade de gerar uma lista de compras com os itens que necessitam de reposição, sugerindo a quantidade necessária e separando os itens essencias dos itens complementares, facilitando o planejamento das compras.

Conclusão:

O Gestor de Dispensa é uma solução prática e eficiente para quem busca manter sua despensa organizada e otimizar o controle de estoque em casa. Com funcionalidades que combinam usabilidade e praticidade, o app oferece uma maneira rápida e eficaz de garantir que itens essenciais não faltem, além de fornecer uma visão detalhada do que está disponível e do que precisa ser reposto. É uma ferramenta útil para qualquer pessoa que deseje manter o controle sobre os itens de sua casa de forma ágil e simples.

🔗 Acesse o repositório do projeto https://github.com/WilliamAlvesOliveira/Gestor-de-Dispensa

Projeto Art Gallery

O projeto Art Gallery foi desenvolvido com o objetivo de criar uma plataforma interativa e envolvente para a exibição de imagens. Cada seção do site foi cuidadosamente projetada para aplicar diferentes técnicas de exibição, proporcionando uma experiência única para o usuário. As imagens são organizadas de acordo com seu estilo e tema, permitindo que o visitante explore uma variedade de representações artísticas, cada uma com sua própria forma de apresentação visual. Além disso, o layout foi pensado para destacar o caráter individual de cada obra, criando uma sensação de imersão na arte ao mesmo tempo em que oferece uma navegação intuitiva e dinâmica.

🔹 Tecnologias utilizadas: HTML, CSS, JavaScript, React

✔️ Layout responsivo para diferentes dispositivos

✔️ Design moderno e intuitivo

🔹 Funcionalidades:

O site conta com um carrossel de imagens que realiza trocas automáticas, além de botões interativos para seleção, proporcionando uma navegação dinâmica e agradável entre as imagens.

Há uma seção dedicada a imagens de animais e plantas nativas do Brasil, que utiliza um layout Masonry para otimizar o espaço e adaptar o tamanho das imagens, garantindo uma visualização harmônica e uma experiência visual agradável.

Na RPG Cards, cada personagem é apresentado com uma imagem de fundo e informações que são atualizadas dinamicamente. Os dados mudam de acordo com a interação do usuário, enquanto a exibição varia conforme o espaço disponível e a orientação da tela, tornando a navegação fluida e adaptável.

Além disso, o site apresenta cards modulares que exibem imagens e trechos de músicas. O posicionamento desses cards é gerado aleatoriamente a cada visita, oferecendo uma experiência única e diversificada para o usuário a cada interação.

A seção Rocker simula uma seleção de personagens no estilo videogame. Você pode escolher um personagem, fazendo com que sua área ganhe destaque visual, e em seguida confirmá-lo, ativando um efeito sonoro e visual de seleção.

O componente Magic 3D World é uma seção encantada que traz fotos que "saltam" da tela com um efeito 3D ao serem clicadas. Você pode girar os cards para explorar o efeito, que é complementado por um brilho especial criado com canvas — adicionando uma camada extra de fofura e magia à experiência.

Conclusão:

O projeto Art Gallery foi desenvolvido para proporcionar uma experiência visual única, onde a diversidade de técnicas de exibição de imagens cria uma interação envolvente e dinâmica com o usuário. Com seções adaptativas, como o carrossel, o layout Masonry e os cards modulares, o site oferece uma experiência personalizada e imersiva a cada visita, tornando-se uma verdadeira galeria de arte digital. A combinação dessas funcionalidades não só exibe conteúdo de forma eficaz, mas também oferece uma experiência de navegação interativa e agradável, refletindo criatividade e inovação.

🔗 Acesse o projeto https://williamalvesoliveira.github.io/art-gallery/
